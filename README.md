## 📋 Project Overview

This is a personal QA practice project where I performed end-to-end manual testing of an e-commerce web application. The goal was to simulate a real-world QA workflow using industry-standard tools.

---

## 🎯 Objectives

- Practice real-world manual testing techniques
- Improve skills in test case design and bug reporting
- Gain hands-on experience with Jira and Postman
- Document testing process in a professional format

---

## 🧪 Testing Types Performed

| Testing Type | Description |
|---|---|
| Functional Testing | Verified all features work as expected |
| Regression Testing | Ensured new changes don't break existing features |
| UI/UX Testing | Checked visual consistency and responsiveness |
| API Testing | Tested REST APIs using Postman |
| Smoke Testing | Quick checks on critical functionalities |
